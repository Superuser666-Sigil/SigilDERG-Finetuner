# Operational Runbooks

This directory contains runbooks for common operational procedures.

## Index

| Runbook | Description |
|---------|-------------|
| [training-execution.md](training-execution.md) | Running training jobs |
| [evaluation-workflow.md](evaluation-workflow.md) | Evaluating trained models |
| [troubleshooting.md](troubleshooting.md) | Common issues and solutions |

## When to Use Runbooks

Runbooks provide step-by-step procedures for:

- **Routine Operations**: Regular training and evaluation runs
- **Incident Response**: Troubleshooting failures and errors
- **Onboarding**: Getting new team members up to speed

## Contributing

When adding a new runbook:

1. Use clear, numbered steps
2. Include expected outputs
3. Document failure modes
4. Add troubleshooting sections
